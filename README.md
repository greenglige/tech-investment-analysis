# tech-investment-analysis

Comparative investment analysis of AAPL and MSFT for Stanford Portfolio Management course.

## Overview
This project provides a detailed analysis comparing the investment potential of Apple Inc. (AAPL) and Microsoft Corporation (MSFT) using various financial metrics.

## Features
- In-depth financial analysis
- Visual representations of data
- Historical stock price comparisons

## Tech Stack
- Python
- Pandas
- Matplotlib

## Usage
Clone the repository and run the analysis script with:
```bash
python analyze.py
```
